new - Mulai permainan baru 
join - Bergabung dengan game saat ini 
start - Mulai permainan
leave - Tinggalkan game yang Anda mainkan 
close - Tutup lobi permainan 
open - Buka lobi permainan
kill - Hentikan permainan 
kick - Tendang pemain keluar dari game
enable_translations - Aktifkan multi-terjemahan 
disable_translations - Nonaktifkan multi-terjemahan 
skip - Lewati pemain saat ini 
notify_me - Dapatkan pesan jika ada game baru di grup ini 
help - Bagaimana cara menggunakan bot ini?
modes - Penjelasan mode permainan
settings - Bahasa dan pengaturan lainnya 
stats - Tampilkan statistik
source - Lihat informasi sumber 
news - Semua berita tentang bot ini
